Do you want to write python code directly in HTML? Well, you can do that with <a href='https://pyscript.net/' target='_blank'> PyScript</a>.

How you ask? Just add these two lines inside the `<head>` tag of your HTML file like this:

```html
<html>
  <head>
    <link rel="stylesheet" href="https://pyscript.net/alpha/pyscript.css" />
    <script defer src="https://pyscript.net/alpha/pyscript.js"></script>
  </head>
</html>
```
